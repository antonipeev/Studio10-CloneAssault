FPS Platformer Game for COMP397 Published by Group/Studio 10.

CURRENT FEATURES
- Player Movement (Sprint, jump crouch, slide)
- Primary / Secondary Weapons (rifle, pistol)
- FPS Camera
- Zooming
- Mouse Aiming and Shooting
- Main Menu
- Pause Menu
- Game Over Menu
- Crosshair
- Enemy AI + Path logic
- Music + Basic Sounds
